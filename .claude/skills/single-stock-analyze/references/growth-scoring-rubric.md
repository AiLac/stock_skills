# 评分锚点（0-5 判分标准）

成长投资视角下，每个因子按 0-5 打分（可给小数）。下面给每个因子的 **0 / 3 / 5** 三档具象描述：0=糟糕、3=中性/合格、5=优秀；1、2、4 按相邻档插值。惩罚项按 0-5 打"利空严重度"（×2 倍率压低总分），红线是布尔触发（命中即封顶）。**所有键名以 `scripts/growth_scorecard.py` 为准**，本文件与之逐字一致。打分前先按 `references/growth-analysis-workflow.md` 完成取证；每条评分需有证据支撑并标分级（强/中/弱/待核实，见 serenity `references/evidence-ladder.md`）。

## 因子锚点（质量分，前 6 项，满分 80）

### revenue_growth_durability（收入增速与持续性，权重 18）

- **0**：收入零增长或下滑；或高增速明显由一次性/拉动透支（pull-forward）支撑，下季度可见放缓，持续性差。
- **3**：稳健中速增长（约 15-30% YoY），趋势平稳，驱动因素清楚但跑道有限，能看到几个季度的能见度。
- **5**：高速增长（约 40%+ YoY）且在加速或可长期维持，多引擎驱动、订单/指引持续上修，增长能见度强。

### tam_penetration（市场空间 / 渗透率，权重 14）

- **0**：市场小或已饱和，渗透率高、跑道短；或 TAM 测算虚高、假设站不住。
- **3**：TAM 够大、渗透率中等，还有数年增长空间，测算口径合理。
- **5**：TAM 巨大且自身在扩张，当前渗透率低、跑道极长，公司有清晰路径吃下更多份额（自下而上测算可验证）。

### moat_chain_position（护城河 / 产业链位置，权重 14）

- **0**：没有壁垒，产品同质、易被替代，处在产业链可有可无的一环，定价权弱。
- **3**：有一定壁垒（品牌/规模/转换成本之一），竞争位置稳定但非稀缺层，能守住份额。
- **5**：壁垒硬且多重（技术+生态+规模/网络效应），卡在稀缺层/卡点上——客户扩产或迭代绕不过它，供应商少、认证慢，定价权强（借 serenity 稀缺层思路）。

### unit_economics_profit_path（单位经济性 / 盈利路径，权重 14）

- **0**：毛利率低且在恶化，烧钱无止境，FCF 持续为负、现金跑道紧张，盈利路径不清（或靠 SBC 粉饰）。
- **3**：毛利率健康且稳定，经营杠杆开始显现，FCF 接近转正或小幅为正，盈利路径清晰可达。
- **5**：高毛利且在扩张，经营杠杆强（收入增长时利润率持续改善），FCF 充沛、自我造血，SBC 占比可控。

### product_customer_momentum（产品与客户势能，权重 12）

- **0**：产品老化、客户流失、净留存 <100%，订单/backlog 萎缩，势能向下。
- **3**：产品迭代正常，客户数稳定增长，净留存约 100-110%，订单平稳。
- **5**：新品强势放量、路线图领先，净留存 >120%、大客户持续扩单，RPO/backlog/billings 同步走强，势能强劲。

### management_capital_allocation（管理层与资本配置，权重 8）

- **0**：管理层信誉差（屡次画饼不兑现）、乱并购/乱花钱，稀释严重而无回报，资本配置毁灭价值。
- **3**：管理层称职、言行基本一致，资本配置中规中矩（合理 Capex、克制并购）。
- **5**：管理层有远见且诚实、长期兑现记录好，资本配置纪律强（高 ROIC 再投资 / 适时回购 / 并购创造价值），与股东利益一致。

## 因子锚点（买点分，后 2 项，满分 20）

### valuation_growth_match（估值与增长匹配度，权重 12）

- **0**：估值与增长严重不匹配——增速一般却给了极高 P/S，或隐含预期高得离谱，明显贵。
- **3**：估值与增长大体匹配，相对自身历史与同业处于合理区间，没有明显高估或低估。
- **5**：估值相对增长明显有吸引力——高增长却给了合理甚至偏低的 P/S（PEG 类友好），相对历史/同业折价，安全边际好。

### entry_expectations（买点 / 预期面，权重 8）

- **0**：预期门槛极高、利好已被充分 price-in，市场拥挤一致看多，财报前高 IV + put skew 显示风险大，当前是差买点。**或：基本面持续 beat 但股价滞涨/回调（距高点回撤震荡）、且远期一致预期/目标价被下修——市场在重估增长持续性或利润率可防御性，完美已透支，属差买点。**
- **3**：预期中性，距一致预期不远，市场看法分化，买点一般（可分批）。股价与基本面动能大体同步、无明显背离。
- **5**：预期门槛低 / 刚被 reset，市场过度悲观或忽视，现价隐含的增长假设保守，是较好的建仓窗口。**或：基本面未坏、悲观预期被过度计入、股价已先跌透（正向背离）——错杀的较好买点。**

> **背离判分护栏**：股价/期权是中/弱证据。本因子要的是**对背离的解释**（市场在边际上重估什么），不是按涨跌机械给分；"股价跌了"本身不构成低分，"完美预期已透支 + 远期假设正被下修"才构成。正反两侧都要用——别只在下跌时扣分、上涨时不审视预期是否已透支。

## 惩罚项判分（×2 倍率）

每个惩罚项按 0-5 打"利空严重度"：**0=无此问题；3=中等利空（需关注，部分 price-in）；5=严重利空（实质压制估值/基本面）**。最终扣分 = 评分 × 2。红线判定与惩罚分开（见下一节）。资金面/期权类信号属中/弱证据，拿不到标"待核实"，详见 `references/risk-and-bear-case.md` 与 `references/positioning-and-options.md`。

- **dilution_financing**：0=无增发/稀释风险，股本稳定或净减少；3=有 ATM/shelf(S-3) 或温和稀释，SBC 部分抵消回购；5=大额增发/配股在即或持续高稀释，股东被严重摊薄。
- **customer_concentration**：0=客户高度分散；3=前几大客户占比偏高（约 20-40%）但有粘性；5=单一/少数客户占比过高（>40% 或单客户主导），一旦流失即重创。
- **accounting_quality**：0=会计干净、现金流与利润匹配；3=存在需留意的项（SBC 偏高、应收/存货略快于收入、一次性项偏多）；5=明显背离（billings 与收入背离、激进确认、频繁重述/调整），质量存疑。
- **governance**：0=治理规范、董事会独立、信息披露透明；3=有瑕疵（双层股权偏激进、关联交易、激励错配）；5=严重问题（管理层诚信差、内控失效、重大利益冲突）。
- **competition_disruption**：0=竞争格局稳定、自身领先；3=竞争加剧或有潜在技术变化，份额/定价承压；5=面临颠覆性技术或强对手降维打击，护城河被实质侵蚀。
- **regulation_geopolitics**：0=无重大监管/地缘风险；3=有政策/出口管制/合规不确定性，影响可控；5=面临实质监管打击、出口禁令、地缘断供等可能伤及核心业务。
- **liquidity_bubble**：0=估值理性、流动性好；3=估值偏高、对利率/风险偏好敏感（高估值最怕逆转）；5=泡沫化估值 + 流动性差/高波动，回撤风险极大。
- **lockup_insider_supply**：0=无近期解禁、无内部人异常卖出、无降评级；3=有解禁日临近 / 个别内部人减持 / 个别降评级，供给压力中等；5=大规模解禁在即（占流通盘高）+ 高管集体减持（Form 4 cluster）+ 降评级潮，筹码面利空叠加。

## 红线 (kill-switch)

红线是布尔（true/false），**命中任意一条即把结论封顶为"回避 / 观望（红线封顶）"**，无论原始分多高——不让高分掩盖致命问题。红线判定**需强证据**（监管文件、官方公告、确凿披露）；仅凭弱证据不得封顶，但应标"待核实"提示。详见 `references/risk-and-bear-case.md`。

- **accounting_fraud_suspicion**：出现重大财务造假嫌疑的硬信号——SEC/监管立案或正式调查、审计师辞任 / 出具非标意见、财报重大重述（restatement）、做空机构提出且公司无法证伪的实质造假指控、关键财务数据被证实系统性失真。
- **core_customer_loss**：核心客户（贡献收入/利润的主力）确认流失或大幅砍单——官方披露/客户公告证实核心大客户终止合作、自研替代、转单竞品，且无可替代的对冲，直接打断增长根基。
- **growth_engine_broken**：增长引擎失效——核心增长业务收入由增转降、连续季度大幅放缓且无回升迹象、全年指引大幅下修并撤回中长期模型、主力产品需求结构性坍塌，成长逻辑被证伪。
