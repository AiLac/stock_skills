# 筹码面与期权市场

这一文件管"供给/筹码面"（谁在卖、要供出多少新股）和"期权市场怎么看"（市场在 price 什么波动/方向）。它们回答的是**"市场在交易什么、有没有近期抛压"**，属情绪/持仓面，定为**中/弱证据**——辅助买点与风险判断，不作核心结论主驱动。映射到评分卡两个惩罚项：`dilution_financing`（稀释/融资）与 `lockup_insider_supply`（解禁/内部人卖出/降评级，筹码供给）。结构性风险与红线见 `references/risk-and-bear-case.md`；判分锚点见 `references/growth-scoring-rubric.md`。

**取证原则**：解禁日、增发、Form 4、评级变化、IV/skew 等**必须联网查证，不靠记忆**；部分期权数据（IV 百分位、skew）联网不一定拿得全——拿不到就标"待核实"并给查证路径，绝不编数字。

## 供给与筹码面

扫描"未来会不会有大量新增/被动卖盘"。每项写清规模、时间窗（date window）、来源、证据强度。映射到 `dilution_financing`（稀释/融资类）与 `lockup_insider_supply`（解禁/内部人/降评级类）两个惩罚项（0-5，×2）。

- **解禁日（lockup expiry）** → `lockup_insider_supply`：IPO/增发锁定期到期日、解禁规模（占流通盘 %）、谁解禁（早期投资人/员工/承销商）。规模大、临近、持有人有卖出动机 → 近期抛压，提高判分。
- **增发 / ATM / shelf** → `dilution_financing`：在架的 S-3 shelf 注册、ATM（at-the-market）计划、已宣布或预期的配股/可转债。看摊薄幅度与必要性（缺钱被动增发 vs 趁高主动融资），稀释股东 → 提高判分。
- **内部人卖出（insider selling）** → `lockup_insider_supply`：Form 4 申报、是否 10b5-1 预设计划（计划内卖出信号弱）、是否高管集体/集中减持（CEO/CFO cluster selling 信号更强）。集中、非计划、临财报/利好 → 提高判分。
- **机构降评级 / 减持** → `lockup_insider_supply`：卖方分析师评级下调、目标价下修潮；13F 机构持仓季度环比下降。成片下调 / 龙头机构离场 → 提高判分（属中证据，口径各异）。
- **回购 vs 稀释净额** → `dilution_financing`：股本是真减少，还是回购被 SBC（股权激励）抵消、净股数仍在增加。净稀释 → 提高 `dilution_financing` 判分；真回购+净股数下降 → 减压。

## 期权市场解读

期权市场是"市场对未来波动与方向的定价"，用来读情绪/持仓，**微调买点因子**（财报前高 IV + put skew 可压低 `entry_expectations`，见 `references/valuation-and-entry.md`），**不触发红线**。

- **隐含波动率 IV + IV 百分位**：市场预期波动多大、当前贵不贵（IV 处于自身历史高/低分位）。
- **财报隐含波动（expected move）**：期权 price-in 的下次财报跳空幅度——市场押财报会怎么动。
- **Put/Call 比率**：多空情绪的粗读（极端值才有意义）。
- **偏斜 skew**：下行保护需求。put skew 走高＝避险/恐慌情绪上升。
- **未平仓 OI 集中度 / max pain**：关键行权价、临到期的"磁吸位"。
- **异常期权活动**：大单 / 不寻常活跃——**仅作 lead 级线索**，不单独下结论。
- **期限结构**：催化剂（财报/裁决）前近月相对远月的升水 / 倒挂，看市场把不确定性押在何时。

## 证据分级

筹码面与期权信号属**情绪/持仓面**，按证据梯子（serenity `references/evidence-ladder.md`；兜底分级见下）定级，并据此约束用途：

- **强证据**：SEC 文件本身——Form 4（内部人卖出）、S-3（shelf/增发）、招股书锁定条款（解禁日与规模）。这些是事实，可硬引用。
- **中证据**：卖方评级/目标价变化、13F 机构持仓变动、聚合站的 IV / put-call（口径各异）。
- **弱证据**：异常期权活动、社媒传闻、未经证实的"大资金"叙事——只作 lead，不作主驱动。
- **整体定位**：以上信号**辅助买点与风险，不作核心结论主驱动**；与基本面冲突时以基本面为准。拿不到的（尤其 IV 百分位、skew、历史分位）一律标"待核实"并写明需哪类数据源。

## 数据源

按证据强度找源；拿不到的标"待核实"并写查证路径，绝不编数字。US 源路径可参 serenity `references/market-source-playbook.md`（US 段）；兜底核心源如下：

- **内部人卖出**：SEC EDGAR → Form 4 / Form 5（强证据，含日期、股数、10b5-1 标记）。
- **增发 / shelf / ATM**：SEC EDGAR → S-3 / S-1 / 424B、8-K（融资公告）、10-Q/10-K 的股本与 SBC 附注（强证据）。
- **解禁日历（lockup expiry）**：招股书锁定条款（强）；交易所 / 第三方 IPO 锁定到期日历（中，口径各异，拿不到标"待核实"）。
- **机构降评级 / 目标价 / 13F**：卖方研报与聚合评级站（中证据）；13F 用 EDGAR 机构申报（强但滞后一季度）。
- **期权数据（IV / IV 百分位 / skew / OI / max pain / expected move）**：交易所或期权数据站；IV 百分位、skew、历史分位常需第三方数据，联网不一定拿得全——拿不到标"待核实"并注明需期权数据源。
