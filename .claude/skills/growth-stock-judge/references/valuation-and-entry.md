# 估值与买点（成长股）

成长股的结论不只看"公司好不好"，还要看"现价贵不贵、市场已经 price-in 了多少"。本文件讲成长股怎么估值、怎么判买点 / 预期面，并把结论映射到两个买点因子：`valuation_growth_match`（估值与增长匹配度）与 `entry_expectations`（买点 / 预期面）。这两项共 20 分，作用是"价格能影响结论，但不会单凭贵就否掉一家好公司"。

**取证原则**：现价、估值倍数、一致预期、增速等**必须联网查证，不靠记忆**；拿不到的标"待核实"并给查证路径（见下方「数据获取」）。每条结论标证据分级（强/中/弱/待核实，见 serenity `references/evidence-ladder.md`；兜底：强=文件/官方数据，中=可信媒体/卖方，弱=社媒/传闻）。判分锚点统一以 `references/growth-scoring-rubric.md` 为准。

## 估值方法（成长股）

成长股早期常无 P/E（不盈利或盈利失真），故主看收入端与增长匹配，而非静态市盈率。

- **P/S vs 增长**：市销率要放到增速里看——40%+ 增速给 15× P/S，和 15% 增速给 15× P/S，含义完全不同。先问"这个 P/S 配得上这个增速吗"。
- **PEG 类（增长调整后的估值）**：用 P/S÷增速 或 P/E÷增速 的直觉做粗筛——同等估值下增速越高越划算；增速放缓时高倍数最危险。
- **EV/Sales（含 EV/NTM Sales）**：用企业价值（含净现金/债务）÷收入，比 P/S 更干净；优先用**前瞻（NTM/明年）**收入，因为买的是未来。
- **Rule of 40**：收入增速 + 利润率（或 FCF margin）≥40 视为健康；越高越能支撑溢价，越低则高估值越站不住。
- **vs 自身历史**：当前倍数处在自己过去 3-5 年区间的高位还是低位（如 EV/NTM Sales 的历史分位），是判断"相对贵/便宜"的强锚。
- **vs 同业**：与同赛道、同增速、同盈利质量的可比公司横向对比倍数，剔除增速差异后看溢价/折价是否合理。
- **隐含预期 / 反向 DCF 直觉**：从现价反推市场"已经假设了"多高的增速/利润率/持续期。若反推出的假设比公司能兑现的还激进，就是贵；若现价隐含的假设保守、留有余地，则有安全边际。这是连接「估值」与「预期面」的桥。

## 买点 / 预期面

同一只股票，"好公司"不变，但买点会随价格与预期高低而变。这一节判"现在是不是好的进场点"。

- **距一致预期的距离**：现价对应的增长/盈利假设，与卖方一致预期相比是偏乐观还是偏保守；股价已跑在预期前面多远。
- **预期门槛（是否已被 price-in）**：利好是否已被充分定价——"好公司但人人皆知、利好出尽"往往是差买点；预期门槛越高，越容易被一次不及预期击穿。
- **近期是否被 reset**：预期是否刚经历下修 / 估值杀（一致预期下调、倍数大幅压缩）。被过度悲观地 reset、而基本面未坏，常是较好的窗口。
- **市场在为什么付费**：现价溢价是付给"已兑现的增长"，还是付给"尚未兑现的故事/期权价值"；后者兑现风险更高，买点要求更苛刻。
- **情绪/持仓辅助信号**：财报前高 IV + put skew、拥挤的一致看多等，提示进场点风险（属中/弱证据，详见 `references/positioning-and-options.md`），用来微调买点，不作核心驱动。

## 股价行为 vs 基本面：预期在边际上被重估什么（必做）

> **核心原则：好公司 ≠ 当前价位的好股票。** 基本面反映过去与现在，股价反映对未来的预期。"基本面这么好为什么不涨/还跌"——真正要问的不是当下业绩多漂亮，而是**市场对未来的预期在边际上发生了什么变化**。当下的好财报，市场早在几个季度前就已经"买入"过了。这一节强制回答四个问题，结论写进报告第 8 节的「市场在重估什么 / 基本面 vs 股价背离」子块：

1. **已经 price-in 了什么**：用反向 DCF 直觉反推现价隐含的增速/利润率/持续期假设（见上一节）。当一只票被定价到"必须季季完美执行"时，好财报只是兑现预期，任何瑕疵都会被放大。
2. **市场在边际上重估什么**（最关键）：把结构性利空当作**边际再定价催化剂**而非静态清单去读——典型四类：① **增长持续性**（高基数上还能不能维持高增速）；② **利润率可防御性**（自研替代/价格战会不会侵蚀毛利与定价权）；③ **需求天花板**（某市场被砍/见顶，削弱"需求无上限"故事）；④ **大客户议价权**（少数客户占比高、且在自研替代品，天平在变）。
3. **股价行为 vs 基本面是否背离**：现价是顺着基本面动能（beat / 指引上调 / TAM 扩张）走，还是背离（基本面强、股价滞涨/回调/距高点回撤震荡）？背离往往说明**远期一致预期/隐含假设正被下修**，或预期已透支——也可能只是噪音，要区分。
4. **背离是否合理 / 何时收敛**：要发生什么（毛利率是否因竞争见顶回落、下一代产品节奏、大客户 capex 是否还在加码、监管落地）才能让股价与基本面重新收敛——这几个才是决定它重回右侧的变量，列进报告第 10 节催化/日历。

**护栏**：价格/期权是**中/弱证据**，这一节的作用是**逼出对背离的解释与对远期假设的检验**，不是按涨跌给分；**不要让"股价跌了"机械地拉低分数**，也不要把 skill 变成追涨杀跌。背离与基本面冲突时仍以基本面为准。**正反两侧都要用**：基本面差但悲观预期被过度计入、股价已先跌透 = 正向再定价（买点变好）；基本面好但完美预期已透支、股价滞涨 = 负向再定价（买点变差）。

*微例*：一家公司当季 +85% 增长、毛利创纪录，股价却从高点回调约 13% 后震荡——这通常不是"市场不认基本面"，而是市场在用更挑剔的眼光重新定价**增长的持续性与利润率的可防御性**（如自研 ASIC 蚕食推理份额、某地缘市场被砍、两大客户占收三成且在自研）。该写的不是"业绩好所以买"，而是"完美已被定价，盯毛利见顶/下一代节奏/大客户 capex 三个收敛变量"。

## 如何映射到评分

把上面两节的结论落到两个买点因子（0-5，可给小数）。**完整 0/3/5 锚点见 `references/growth-scoring-rubric.md`，此处只给映射逻辑，避免重复维护**。

- **valuation_growth_match（估值与增长匹配度，权重 12）**：由「估值方法」一节驱动——P/S/EV-Sales/PEG 类/Rule of 40 与增速是否匹配、相对自身历史与同业是否合理。匹配甚至偏便宜→高分；严重不匹配（增速一般却给极高倍数、隐含预期离谱）→低分。
- **entry_expectations（买点 / 预期面，权重 8）**：由「买点 / 预期面」与「股价行为 vs 基本面」两节共同驱动——预期门槛高低、是否已 price-in、是否刚被 reset，**以及基本面与股价是否背离**。门槛低 / 刚 reset / 过度悲观 / 股价已先跌透→高分；门槛极高 / 利好出尽 / 拥挤一致看多 / 基本面持续 beat 但股价滞涨回调且远期一致预期被下修→低分。**判分必须连同背离的解释一起写**（"市场在重估增长持续性/利润率可防御性，故买点偏差"），不机械按价格涨跌打分。
- **极端利空联动**：纯粹"贵"只压低这两项，不触红线；但若属泡沫化估值 + 流动性差，另计入 `liquidity_bubble` 惩罚项（见 `references/risk-and-bear-case.md`）。买点因子不触发红线。

## 数据获取

按证据强度找源；拿不到的标"待核实"并写明查证路径，绝不编数字。

- **现价 / 历史价格**：交易所或行情站实时报价；现价**必须联网查**，不靠记忆。（强证据=交易所/官方行情）
- **收入与增速 / 前瞻收入**：最新 10-Q / 10-K（SEC EDGAR）取已实现收入；前瞻（NTM）收入用公司指引 + 卖方一致预期。US 源路径见 serenity `references/market-source-playbook.md`（US 段：SEC 文件、transcript、investor presentation；兜底=本目录列核心源）。
- **P/S、EV/Sales、Rule of 40**：用现价/市值/企业价值（含净现金、债务）÷收入自算，分子分母都标口径与日期；自算可控、避免站点口径不一。
- **一致预期 / 目标价 / 评级**：卖方一致预期与目标价多为聚合数据站（中证据，口径各异），拿不到标"待核实"。
- **历史估值分位（如 EV/NTM Sales 分位）**：常需第三方数据站，联网不一定拿得全——拿不到标"待核实"并注明需第三方估值数据源。
- **隐含预期 / 反向 DCF**：用现价 + 可得增速/利润率假设手算近似，明确标注为"基于假设的测算"而非硬数据。
- **IV / skew 等情绪面**：见 `references/positioning-and-options.md`（属中/弱证据，常拿不全，标"待核实"）。
