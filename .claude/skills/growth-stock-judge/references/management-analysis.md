# 管理层 / 领导力分析（定性，不打分）

从"**人**"的维度辅助判断公司发展潜力。这是**定性**模块——不产出任何因子分；判断写进报告「👤 管理层 / 领导力」节，并可**定性联动现有评分键**（不新增键）。`management-analyst` 子代理按本文件取证。

**来源（按强弱）**：强 = DEF 14A 代理声明（薪酬/持股/董事会构成）、10-K / 20-F、Form 4（内部人交易）、财报电话会 transcript、S-1/招股书的管理层段；中 = 管理层公开访谈/演讲、可信媒体人物报道、卖方对管理层的评述；弱 = Glassdoor 等员工评价、社媒。每条结论标 强/中/弱/待核实，取证失败按 SKILL.md「🔁 闭环核实协议」换路再取。

## 1. 创始人 / CEO 背景与往绩

- 核心高管是谁（CEO/创始人/CFO/CTO），出身、履历、任期多久。
- 是否**创始人主导（founder-led）**——创始人是否仍在位、持股、掌舵。
- 过往经营/创业成绩：之前带过什么公司/业务，做成还是做砸；在本公司任内的关键决策结果。
- → 红旗：频繁更换 CEO、空降且无相关履历、past 有重大失败/失信记录。

## 2. 愿景与战略执行力（said-vs-done）

- 长期愿景是否清晰、可信、与资源匹配（不是 PPT 画饼）。
- 战略连贯性：方向是否反复横跳、追风口。
- **承诺兑现率（said-vs-done）**：过去 2-3 年公开承诺过的目标/路线图/指引，实际兑现了多少。
- → 红旗：反复跳票、画饼不兑现、战略一年一变。

## 3. 利益绑定（skin in the game）

- 内部人/创始人持股比例（按 DEF 14A / Form 4）；是在增持还是持续减持。
- 薪酬结构：是否对齐**长期价值**（多年期、与运营里程碑挂钩）还是短期股价/无条件大包。
- 关联交易、自我交易、把公司当提款机的迹象。
- → 联动 `governance` 惩罚；重大利益不一致是扣分项。

## 4. 团队稳定性 / 关键人物风险

- 高管流失/出走频率（CFO/核心技术/业务负责人离职潮是预警）。
- **关键人物依赖（key-person）**：公司是否过度系于一个人；此人若离开影响多大。
- 接班与板凳深度；文化与人才吸引/保留（弱证据：Glassdoor、招聘/离职信号）。
- → 关键人物突然出走且无接班，可作 `growth_engine_broken` 风险输入（由综合代理判定）。

## 5. 对发展潜力的净判断 + 联动现有键

- 一句话**净判断**：管理层是发展潜力的**加分项**还是**扣分项**，关键人物风险有多大。
- **联动现有键（不新增因子）**：
  - 资本配置往绩（与 earnings-analyst 重叠）→ 贡献 `suggested_factor_scores.management_capital_allocation`，综合代理按证据择优调和。
  - 利益不绑定 / 治理失当 → `suggested_penalties.governance`。
  - 重大诚信/造假（需强证据）→ `suggested_red_lines.accounting_fraud_suspicion`；核心关键人物出走 → `growth_engine_broken` 风险输入。
- 对结论框的影响：写清"创始人主导、执行力强 → 提升长期信心" 或 "治理存疑 / 关键人物风险高 → 压低信心"。
